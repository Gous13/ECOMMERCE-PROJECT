## 💡 Getting Started

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Gous13/ECOMMERCE-PROJECT.git
   ```
2. **Navigate to the project directory:**
   ```bash
   cd ECOMMERCE-PROJECT
   ```
3. **Install dependencies:**
   ```bash
   npm install
   ```
4. **Start the development server:**
   ```bash
   npm start
   ```
